Bank Loan Analysis 
Overview

This project focuses on analyzing bank loan data to uncover key insights related to loan performance, customer behavior, and financial risk. Using SQL, Power BI, and Python (Jupyter Notebook), the project provides a comprehensive analysis of loan applications, funded amounts, repayments, and risk segmentation.

The goal of this project is to support data-driven decision-making by identifying trends, evaluating loan quality, and monitoring financial performance.

Tools & Technologies

SQL – Data querying and KPI extraction

Power BI – Interactive dashboard and data visualization

Python (Jupyter Notebook) – Data preprocessing and analysis

Excel/CSV – Dataset handling

📂 Dataset

The dataset contains detailed information about loan applications, including:

Loan amount, total payment

Interest rate and DTI (Debt-to-Income ratio)

Loan status (Fully Paid, Current, Charged Off)

Customer details (state, employment length, home ownership)

Loan purpose and term

🔗 Full Dataset:
Dataset is included in this repository.

📊 Key KPIs Analyzed

Total Loan Applications

Total Funded Amount

Total Amount Received

Average Interest Rate

Average DTI (Debt-to-Income Ratio)

Good Loan vs Bad Loan Analysis

🔍 Key Insights

Identified good loans (Fully Paid, Current) and bad loans (Charged Off) to evaluate risk.

Analyzed monthly trends in loan applications and repayments.

Evaluated state-wise performance to identify high and low-performing regions.

Studied customer segments based on employment length, home ownership, and loan purpose.

Compared funded vs received amounts to track financial performance.
